# 🧠 AI Hallucination Detector

A full-stack system that answers user questions using an LLM, automatically **splits the answer into individual factual claims**, retrieves **trusted evidence** for each claim (Wikipedia-based retrieval), and uses an **NLI (Natural Language Inference) transformer model** to classify each claim as `TRUE`, `FALSE`, or `UNSURE` — producing a hallucination score, explanation, and sources for every response.

```
User Question
   ↓
LLM Engine (Groq API)  →  Raw Answer
   ↓
Claim Splitter          →  [claim 1, claim 2, ...]
   ↓
Retrieval (Wikipedia)   →  Trusted evidence per claim
   ↓
NLI Verifier (DeBERTa)  →  TRUE / FALSE / UNSURE + confidence
   ↓
Hallucination Score + Explanation + Sources
   ↓
Frontend Dashboard (chat UI, meter, per-claim graph, sources)
```

## ✨ Features

- **LLM-powered chatbot** (Groq API — free tier, OpenAI-compatible)
- **Automatic claim splitting** of any LLM response
- **RAG-lite retrieval** against Wikipedia for trusted evidence
- **NLI-based fact verification** (`cross-encoder/nli-deberta-v3-xsmall`) — detects entailment/contradiction, not just surface similarity
- **Hallucination scoring** (0–100%) per claim and overall
- **Full dashboard**: hallucination meter, confidence bar chart (Chart.js), source links, claim-by-claim breakdown
- **Persistent history** via SQLite
- **Mock mode**: works out of the box with zero API key (canned demo answers) so anyone can clone and run it instantly

## 🚀 Setup & Run

### 1. Clone and install

```bash
git clone <your-repo-url>
cd hallucination-detector/backend
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. (Optional) Add a free Groq API key

Without a key the app runs in **mock mode** (canned answers, full pipeline still works for demo).
For real LLM answers:

```bash
cp .env.example .env
# then edit .env and paste your key from https://console.groq.com
```

### 3. Run the server

```bash
uvicorn main:app --reload
```

### 4. Open the app

Visit **http://localhost:8000** — the frontend is served directly by FastAPI.

> First run will download the NLI model (~300MB) automatically via `sentence-transformers` — this can take a minute.

## 🔌 API Endpoints

| Method | Endpoint    | Description                                  |
|--------|-------------|-----------------------------------------------|
| POST   | `/chat`     | Send a question, get raw LLM answer + chat_id |
| POST   | `/verify`   | Verify a chat_id's answer, get hallucination report |
| GET    | `/history`  | List past chats with scores                   |
| GET    | `/health`   | Health check + mock mode status               |

Interactive API docs available at **http://localhost:8000/docs** (Swagger UI, auto-generated by FastAPI).

### Example

```bash
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"question": "Who won FIFA 2022?"}'

# → {"chat_id": 1, "question": "...", "answer": "..."}

curl -X POST http://localhost:8000/verify \
  -H "Content-Type: application/json" \
  -d '{"chat_id": 1}'
```
